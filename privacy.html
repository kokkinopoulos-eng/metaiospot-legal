🌤 MetAIoSpot — Πολιτική Απορρήτου

Τελευταία ενημέρωση: 26 Μαΐου 2026

Σύνοψη: Η MetAIoSpot δεν συλλέγει, δεν αποθηκεύει και δεν μοιράζεται προσωπικά δεδομένα χρηστών σε κανέναν server.

1. Ποιοι είμαστε

Η εφαρμογή MetAIoSpot αναπτύχθηκε από τον BabKok (webdevelopment.gr). Επικοινωνία: MetAIoSpot@webdevelopment.gr

2. Δεδομένα που χρησιμοποιούνται τοπικά

Η εφαρμογή χρησιμοποιεί τα παρακάτω δεδομένα αποκλειστικά στη συσκευή σας:

  • Τοποθεσία GPS: Για εμφάνιση τοπικού καιρού και χάρτη. Δεν αποστέλλεται σε servers μας.
  • API Keys: Αποθηκεύονται κρυπτογραφημένα στη συσκευή (flutter_secure_storage). Δεν αποστέλλονται πουθενά εκτός από τους αντίστοιχους παρόχους (Google/Anthropic/OpenAI).
  • Ρυθμίσεις εφαρμογής: Αποθηκεύονται τοπικά (SharedPreferences).

3. Τρίτες υπηρεσίες

Η εφαρμογή επικοινωνεί με τις παρακάτω εξωτερικές υπηρεσίες:

  • Open-Meteo API (open-meteo.com): Δεδομένα καιρού. Αποστέλλονται μόνο συντεταγμένες GPS.
  • OpenStreetMap (openstreetmap.org): Tiles χάρτη. Αποστέλλονται μόνο συντεταγμένες.
  • Google Geocoding API: Μετατροπή συντεταγμένων σε όνομα τοποθεσίας.
  • Google Gemini API (προαιρετικό, BYOK): Αν ο χρήστης παρέχει δικό του API key, οι ερωτήσεις αποστέλλονται στην Google. Υπόκειται στην Πολιτική Απορρήτου Google.
  • Anthropic Claude API (προαιρετικό, BYOK): Υπόκειται στην Πολιτική Απορρήτου Anthropic.
  • OpenAI ChatGPT API (προαιρετικό, BYOK): Υπόκειται στην Πολιτική Απορρήτου OpenAI.

4. Δικαιώματα πρόσβασης

  • LOCATION: Για εντοπισμό τοποθεσίας και εμφάνιση καιρού.
  • CAMERA: Μόνο αν ο χρήστης επιλέξει ανάλυση φωτογραφίας ουρανού μέσω AI.
  • INTERNET: Για λήψη δεδομένων καιρού και AI επικοινωνία.

5. Δεδομένα παιδιών

Η εφαρμογή δεν απευθύνεται σε παιδιά κάτω των 13 ετών και δεν συλλέγει εν γνώσει της δεδομένα παιδιών.

6. GDPR — Δικαιώματά σας

Καθώς δεν συλλέγουμε προσωπικά δεδομένα σε servers μας, δεν υπάρχουν δεδομένα προς διαγραφή ή εξαγωγή από εμάς. Για δεδομένα που αποστέλλονται στις τρίτες υπηρεσίες, επικοινωνήστε απευθείας με αυτές.

7. Αλλαγές στην Πολιτική

Τυχόν αλλαγές θα δημοσιεύονται σε αυτή τη σελίδα. Συνιστούμε περιοδική επίσκεψη.
