🌤 MetAIoSpot — Πολιτική Απορρήτου

Τελευταία ενημέρωση: 29 Μαΐου 2026

Σύνοψη: Η MetAIoSpot δεν συλλέγει, δεν αποθηκεύει και δεν μοιράζεται προσωπικά δεδομένα χρηστών σε δικούς μας servers. Η εφαρμογή χρησιμοποιεί Google AdMob για διαφημίσεις, το οποίο ενδέχεται να συλλέγει δεδομένα σύμφωνα με την πολιτική της Google.

1. Ποιοι είμαστε

Η εφαρμογή MetAIoSpot αναπτύχθηκε από τον BabKok (webdevelopment.gr). Επικοινωνία: MetAIoSpot@webdevelopment.gr

2. Δεδομένα που χρησιμοποιούνται τοπικά

Η εφαρμογή χρησιμοποιεί τα παρακάτω δεδομένα αποκλειστικά στη συσκευή σας:

  • Τοποθεσία GPS: Για εμφάνιση τοπικού καιρού, χάρτη και εύρεση κοντινών παραλιών. Δεν αποστέλλεται σε servers μας.
  • API Keys: Αποθηκεύονται κρυπτογραφημένα στη συσκευή (flutter_secure_storage). Δεν αποστέλλονται πουθενά εκτός από τους αντίστοιχους παρόχους (Google/Anthropic/OpenAI).
  • Ρυθμίσεις εφαρμογής: Αποθηκεύονται τοπικά (SharedPreferences).
  • Αγαπημένες τοποθεσίες: Αποθηκεύονται τοπικά στη συσκευή σας.

3. Τρίτες υπηρεσίες

Η εφαρμογή επικοινωνεί με τις παρακάτω εξωτερικές υπηρεσίες:

  • Open-Meteo API (open-meteo.com): Δεδομένα καιρού. Αποστέλλονται μόνο συντεταγμένες GPS.
  • OpenStreetMap / Overpass API: Tiles χάρτη και εύρεση παραλιών. Αποστέλλονται μόνο συντεταγμένες.
  • Google Geocoding API: Μετατροπή συντεταγμένων σε όνομα τοποθεσίας.
  • Google AdMob: Η εφαρμογή χρησιμοποιεί Google AdMob για την εμφάνιση διαφημίσεων. Το AdMob ενδέχεται να συλλέγει δεδομένα συσκευής και διαφημιστικό αναγνωριστικό για εξατομίκευση διαφημίσεων. Υπόκειται στην Πολιτική Απορρήτου Google.
  • Google Gemini API (προαιρετικό, BYOK): Αν ο χρήστης παρέχει δικό του API key, οι ερωτήσεις αποστέλλονται στην Google. Υπόκειται στην Πολιτική Απορρήτου Google.
  • Anthropic Claude API (προαιρετικό, BYOK): Υπόκειται στην Πολιτική Απορρήτου Anthropic.
  • OpenAI ChatGPT API (προαιρετικό, BYOK): Υπόκειται στην Πολιτική Απορρήτου OpenAI.

4. Διαφημίσεις (Google AdMob)

Η εφαρμογή χρησιμοποιεί Google AdMob για εμφάνιση διαφημίσεων. Το AdMob ενδέχεται να χρησιμοποιεί:
  • Διαφημιστικό αναγνωριστικό (Advertising ID) της συσκευής
  • Δεδομένα χρήσης για εξατομίκευση διαφημίσεων
  • Πληροφορίες σύνδεσης δικτύου
Μπορείτε να απενεργοποιήσετε την εξατομίκευση διαφημίσεων από τις ρυθμίσεις της συσκευής σας (Ρυθμίσεις → Google → Διαφημίσεις).

5. Δικαιώματα πρόσβασης

  • LOCATION: Για εντοπισμό τοποθεσίας, εμφάνιση καιρού και εύρεση κοντινών παραλιών.
  • CAMERA: Μόνο αν ο χρήστης επιλέξει ανάλυση φωτογραφίας ουρανού μέσω AI.
  • INTERNET: Για λήψη δεδομένων καιρού, εμφάνιση διαφημίσεων και AI επικοινωνία.

6. Δεδομένα παιδιών

Η εφαρμογή δεν απευθύνεται σε παιδιά κάτω των 13 ετών και δεν συλλέγει εν γνώσει της δεδομένα παιδιών.

7. GDPR — Δικαιώματά σας

Καθώς δεν συλλέγουμε προσωπικά δεδομένα σε servers μας, δεν υπάρχουν δεδομένα προς διαγραφή ή εξαγωγή από εμάς. Για δεδομένα που συλλέγει το Google AdMob ή οι τρίτες υπηρεσίες, επικοινωνήστε απευθείας με αυτές ή χρησιμοποιήστε τις ρυθμίσεις απορρήτου της συσκευής σας.

8. Αλλαγές στην Πολιτική

Τυχόν αλλαγές θα δημοσιεύονται σε αυτή τη σελίδα με ενημερωμένη ημερομηνία. Συνιστούμε περιοδική επίσκεψη.
